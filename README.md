### Getting Started

1. Clone project

```
git clone https://github.com/muzaffercankaplan/DeepControl.git
```

2. Open a new terminal

```
 npm install OR yarn add
```

3. Last, run the development server:

```bash
npm run dev OR yarn dev
```
